ESPECIFICAÇÃO DE CASO DE USO

UC01 – EFETUAR LOGIN

Atores Descrição
Usuário Usuário com permissão de consulta
Administrador do Sistema Usuário com permissão completa no Sistema

Objetivo

    Permitir ao usuário cadastrar-se para adicionar e gerenciar suas agendas de viagens e iniciar o processo de localização de pontos de interesse.

Ator Primário: Cliente

Pré Condições

Não se Aplica

Pós Condições

Não se Aplica

Fluxo Principal

FP - Efeturar Login
Passo Interação do Ator
1 Sistema Apresenta a tela de login
2 Usuário digita e-mail
3 Usuário digita senha
4 Usuário seleciona botão login
5 Sistema Valida informações
6 Sistma exibe tela Ponto de Interesses
7 O caso de uso é encerrado

Fluxo Alternativo

FA01 - Novo Usuário
Passo Interação do Ator
1 Sistema Apresenta a tela de login
2 Usuário digita e-mail
3 Usuário digita senha
4 Usuário seleciona botão criar
5 Sistema verifica informações
6 Sistma redireciona usuários para tela de Ponto de Interesses
7 O fluxo alternativo é encerrado


FA02 - Usuário escolhe conectar-se utilizando conta do Google+
Passo Interação do Ator
1 Sistema Apresenta a tela de login
2 Usuário decide conectar-se utilizando conta do Google+
3 Sistema exibe tela de Login da conta do Google+
4 Usuário informa e-mail e senha da conta do Google+
5 Sistema solicita autorização para utilizar as informações do Google+
6 Sistema recupera informações para que a conta seja habilitada
7 Sistema re-direciona usuário para tela Pontos de Interesse
8 O fluxo alternativo e encerrado
